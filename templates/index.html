text-on-image

This app is an innovative tool for those who want to easily add text to their images. It allows users to customize the text by choosing the font, font size, color and position, and even rotate the text by a specified angle. The user can select an image and add the text on it. The generated image can be saved and used as per user's requirement. With this app, users can now easily add text to their images and make them more personalized and unique.
